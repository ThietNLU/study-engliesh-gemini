Công nghệ

  • React 18, Zustand, Tailwind CSS
  • Firebase: Auth, Firestore, Hosting
  • Google Gemini API (2.0 Flash)
  • CRA, ESLint, Prettier, Husky

Tính năng chính

  • AI Quiz (A1–C2), 7+ dạng: MCQ, Fill blank, TF, Matching, Ordering, Gap fill…
  • Flashcards + SM-2, thống kê tiến bộ
  • Sync đám mây & hoạt động offline
